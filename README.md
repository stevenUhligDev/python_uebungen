# Python Training

Dieses Repository enthält meine Python-Übungen auf meinem Raspberry Pi 5.

## Umgebung

- Betriebssystem: Debian GNU/Linux 13
- Python 3.13
- Editor: VS Code / SSH
- Virtuelle Umgebung_ '.venv'

## Projekt starten

Virtuelle Umgebung aktivieren:

```bash
source .venv/bin/activate

Programm ausführen

python main.py

```text
